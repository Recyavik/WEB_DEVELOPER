Быстрые настройки:
Яндекс.Почта — два обязательных шага:
Шаг 1. mail.yandex.ru → Настройки (⚙) → Почтовые программы → включить IMAP ✓
Шаг 2. Создать пароль приложения:
   → id.yandex.ru → Безопасность → Пароли приложений → Создать
   → Скопировать сгенерированный пароль и вставить в поле ниже
Обычный пароль от ящика для SMTP не подходит — только пароль приложения!

Mail.ru: Настройки → Безопасность → Пароли для внешних приложений
Gmail: Аккаунт Google → Безопасность → Пароли приложений
Для Яндекса — пароль приложения с id.yandex.ru, не пароль от ящика
Пример: MorphoSyntaxTrening <noreply@school.ru> — или оставьте пустым
