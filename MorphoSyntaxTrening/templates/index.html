Уровни тренажёра

Ознакомительный
Определить часть речи каждого слова.
Начальный
Часть речи + начальная форма (лемма / инфинитив).
Базовый
+ непостоянные признаки: наклонение, время, число, падеж, род…
Углублённый
+ постоянные признаки (склонение, спряжение, вид, залог…) + синтаксическая роль.
Продвинутый
Полный морфологический разбор: все постоянные и непостоянные признаки + члены предложения.

Как проходить тренажёр

  1. Войдите по коду учителя, выберите группу и введите пароль
  2. Откройте тренажёр из списка — найдите по названию или ID
  3. Разберите каждое слово согласно уровню задания
  4. Нажмите «Проверить» в любой момент — не нужно разбирать всё
  5. Получите звёзды: 1 балл за слово при верном разборе всех признаков
  6. Смотрите детальный разбор ошибок сразу после сдачи
