# MorphoTrening — Интерактивный тренажёр простого морфосинтаксического анализа предложений

Веб-приложение для отработки навыков определения частей речи. Ученики перетаскивают цветные метки на слова предложения, система автоматически проверяет ответы и сохраняет результаты в журнал.

---

## Роли

| Роль | Пароль входа | Возможности |
|---|---|---|
| **Учитель** | `teacher` | Управление группами и учениками, создание тренажёров, загрузка книги-источника, просмотр и удаление статистики |
| **Ученик** | Индивидуальный (генерируется учителем) | Прохождение тренажёров, просмотр своей статистики |

---

## Технологии

- **Python 3.12** + **Flask 3** — веб-фреймворк
- **Flask-SQLAlchemy** + **SQLite** (WAL-режим) — база данных
- **pymorphy3** — разбор частей речи русского текста
- **pdfminer.six** — извлечение текста из PDF
- **Docker / Docker Compose** — контейнеризация

## Быстрый запуск через Docker (рекомендуется)

### Требования
- [Docker Desktop](https://www.docker.com/products/docker-desktop/) (Windows/Mac/Linux)

### Шаги

```bash
# 1. Перейдите в папку проекта
cd MorphoTrening

# 2. Соберите и запустите контейнер
docker compose up --build

# 3. Откройте браузер
# http://localhost:5000
```

Для запуска в фоновом режиме:
```bash
docker compose up -d --build
```

Остановка:
```bash
docker compose down
```

> **База данных** сохраняется в именованном Docker-volume `db_data` и не теряется при перезапуске контейнера.

## Запуск без Docker (локально)

### Требования
- Python 3.10+
- pip

### Шаги

```bash
# 1. Перейдите в папку проекта
cd MorphoTrening

source venv/bin/activate

pip install -r requirements.txt

# 5. Откройте браузер
# http://localhost:5000
```

## Первые шаги после запуска

1. Откройте **http://localhost:5000**
2. Нажмите **«Войти как учитель»**, введите пароль `teacher`
3. В разделе **«Ученики»** создайте группу и добавьте учеников — система сгенерирует пароли
4. В разделе **«Тренажёры»** создайте тренажёр и добавьте предложения:
   - **Вручную** — введите любое предложение
   - **Из книги** — загрузите книгу в разделе «Книга», затем задайте количество слов и нажмите «Сгенерировать»
5. Выдайте ученикам их пароли — они могут входить и проходить тренажёры

---

## Генерация предложений из книги

1. Войдите как учитель → раздел **«Книга»**
2. Загрузите текстовый файл (`.txt`, UTF-8 или Windows-1251) или PDF-книгу
3. В редакторе любого тренажёра появится панель **«Генерация из книги»**
4. Укажите диапазон слов (например, от 6 до 12) и нажмите **«Сгенерировать»**
5. Просмотрите предложение и добавьте одним нажатием или запросите другое

Система автоматически исключает предложения с прямой речью, вопросительные и предложения без русских слов.

## Части речи и цвета

| Часть речи | Цвет |
|---|---|
| Существительное | Синий |
| Прилагательное | Зелёный |
| Глагол | Красный |
| Наречие | Фиолетовый |
| Местоимение | Янтарный |
| Предлог | Серый |
| Союз | Розовый |
| Частица | Голубой |
| Причастие | Оранжевый |
| Деепричастие | Лаймовый |
| Числительное | Бирюзовый |
| Междометие | Коричневый |

### Особенности разбора (соответствие школьной грамматике)

- **его / её / их** — всегда Местоимение (притяжательные)
- **этот, тот, мой, твой, наш, ваш, какой, который** и др. — Местоимение (pymorphy3 тегирует их как прилагательные, исправлено)
- **первый, второй, третий** и др. порядковые — Числительное (аналогично)

При обновлении логики разбора используйте кнопку **«↻ Пересчитать разбор»** в настройках тренажёра.
